# 项目结构说明

## 目录结构

```
vue3-test-01/
├── public/              # 静态资源（不经过构建）
├── src/                 # 源代码
│   ├── assets/          # 静态资源
│   ├── components/      # 全局组件
│   ├── directive/       # 自定义指令
│   ├── layout/          # 布局组件
│   ├── providers/       # 工具类
│   ├── router/          # 路由配置
│   ├── stores/          # 状态管理
│   └── views/           # 页面组件
├── doc/                 # 项目文档
├── dist/                # 构建输出目录
├── eslint.config.js     # ESLint 配置
├── vite.config.js       # Vite 配置
└── package.json         # 项目配置
```

## 核心目录说明

### src/assets/ - 静态资源

```
assets/
├── icon/           # 图标文件
│   └── file/       # 文件类型图标
├── json/           # JSON 数据文件
├── login/          # 登录页背景图
├── akm.scss        # 全局 SCSS 变量
├── base.css        # 基础样式
└── main.css        # 主样式文件
```

### src/components/ - 全局组件

存放全局可复用的组件，在 `main.js` 中统一注册：

```
components/
├── akm-dialog/       # 弹窗组件
├── akm-form/         # 表单组件
├── akm-form-view/    # 表单详情展示组件
├── akm-radio-dict/   # 字典单选组件
├── akm-select/       # 下拉选择组件
├── akm-select-dict/  # 字典下拉选择组件
├── akm-table/        # 表格组件
├── akm-warn-btn/     # 警告按钮组件
└── index.js          # 组件注册入口
```

### src/directive/ - 自定义指令

```
directive/
├── dialogDrag.js   # 弹窗拖拽指令
├── focus.js        # 自动聚焦指令
├── margin.js       # 外边距指令
├── padding.js      # 内边距指令
├── permission.js   # 权限指令
├── textAlign.js    # 文本对齐指令
└── index.js        # 指令注册入口
```

### src/layout/ - 布局组件

后台管理系统的整体布局：

```
layout/
├── index.vue           # 主布局容器
├── TheAsideMenu.vue    # 侧边菜单

├── TheBreadcrumb.vue   # 面包屑导航
├── TheHeader.vue       # 顶部导航栏
└── TheHeaderMessage.vue# 消息通知组件
```

### src/providers/ - 工具类

```
providers/
├── api.js            # API 接口定义
├── dict.js           # 字典工具
├── eventBus.js       # 事件总线
├── helper.js         # 业务相关工具方法

├── security.js       # 安全加密工具
├── sessionStorage.js # 会话存储封装

│   ├── baseRequestHelper.js# 请求辅助（加密签名）
```

### src/router/ - 路由配置

```

```

```
stores/

├── permission.js   # 权限状态

├── user.js         # 用户状态
```


## 文件命名规范

| 类型     | 命名规则   | 示例                  |
| -------- | ---------- | --------------------- |
| 组件文件 | kebab-case | `akm-table/index.vue` |
| 组件名称 | PascalCase | `AkmTable`            |
| 页面组件 | kebab-case | `login-log/index.vue` |
| 工具类   | camelCase  | `localStorage.js`     |
| Store    | camelCase  | `useUserStore`        |
| 指令     | camelCase  | `v-has`               |
